Behavior Notes

Non-visual · animation · truncation · conditional visibility

Item Row (Compound)

  • Sell button is fully hidden (display:none) when qty === 0 — the layout collapses, leaving just qty + add.
  • Add button receives the disabled attribute when inventory is at the player's capacity limit.
  • The price tag pill uses a flat #fef08a yellow (not a gradient), a rotate(−3deg) tilt, and an inline 🪙 emoji — no separate coin disc.
  • The wooden tray container does not respond to hover as a unit. Only the inner Sell/Add pills fire pointer events.

Category Tab (Awning)

  • Awning height animates via CSS height transition 220ms cubic-bezier(.34,1.56,.64,1) — a springy "pull-down" feel.
  • Tab group behaves as a radio — exactly one tab is active at all times; active label bg is always #fff.
  • The scallop SVG fill must match c1 (stripe color) to appear continuous with the awning.
  • Tab width is fixed at 140px. Labels must not exceed one line; shorten copy before increasing width.

Action Board

  • Boards are hidden (not disabled) when the corresponding need is at max capacity — there's nothing to do.
  • Hover floats the card up (translateY(−3px) scale(1.02)). Press returns to baseline.
  • The wooden rod and brass connector balls are presentational — they have no pointer events and carry no state.
  • Labels wrap to 2 lines at word boundary. The board height grows to accommodate — never clip with overflow:hidden.
